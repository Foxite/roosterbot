ResolveString(CultureInfo, Component, String)

Resolve a string. When the input starts with a # symbol, it is seen as a string resource key and the string except the # is passed into GetString(CultureInfo, String). Otherwise, it is returned as-is. If you want your string to start with a # but don't want it to be resolved, then you can escape it with a . If your string needs to start with # then you can use \# and so on.4
