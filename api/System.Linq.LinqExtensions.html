Duplicates<T>(IEnumerable<T>, Nullable<IEqualityComparer<T>>)

Enumerates all duplicate items in an enumeration. If a duplicate occurs more than twice in an enumeration it will be yielded one time less than the amount of times it occurs in the source enumeration. For example, a duplicate occuring 4 times will be yielded 3 times. You can combine this with .Distinct() to avoid this.
