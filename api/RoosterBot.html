This class stores data about a channel's preferences with this bot. Built-in are the language and command prefix, and it can store arbitrary custom data.

This arbitrary data is serialized using Newtonsoft.Json. This means most built-in .NET types will work, and any custom data structure that you store must be serializable with Newtonsoft.Json. Read its documentation to learn more.

RoosterCommandContext

The object that represents the context in which a command is being exeucted. It persists from message reception to post-command handling, and it holds the message being responded to, as well as the PlatformComponent which received the message that originated this context. It also holds the UserConfig for the User and the ChannelConfig for the Channel, which is often accessed by multiple unrelated classes in the execution process, so that these classes do not have to retrieve the config data independently.
