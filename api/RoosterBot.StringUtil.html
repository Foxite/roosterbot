BreakStringIntoLines(String, Int32)

Break a string up into lines, with a maximum length of each line. The string will be split up into words by space, and each line will contain the maximum amount of words without exceeding the maximum line length. If a single word exceeds the maximum line length, it will be broken up and hyphenated.

FormatTextTable(IReadOnlyList<IReadOnlyList<String>>, Nullable<Int32>)

Generate a table for use in a Discord message. Output will be formatted into a code block. Warning: This will glitch out when the table array is jagged.
